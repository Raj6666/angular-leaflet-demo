Leaflet快速开始

Leaflet是一个开源的JavaScript库，对移动端友好且有很好的交互性。 大小仅仅只有 33 KB， 同时具有大多数地图所需要的特点。 Leaflet设计的非常简单易懂, 同时具有很好的性能和易用性。 它在桌面端和移动端都工作的相当高效，并有大量的插件用于扩张Leaflet的功能。

准备你的页面

在开始写代码之前，你需要完成以下几个步骤来准备你的页面：

  • 在页面的head标签中添加以下代码来引入CSS和JavaScript文件：

    <link href="https://cdn.bootcss.com/leaflet/1.2.0/leaflet.css" rel="stylesheet" type="text/css" />
<script src="https://cdn.bootcss.com/leaflet/1.2.0/leaflet.js" type="text/javascript" ></script>

  • 将id为map的div标签放到地图显示的地方，如body中：

    <div id="map"></div>

  • 【可选】确保地图容器有定义好的高度，例如在css文件中添加如下代码：

    html, body,#map {height: 100%;width: 100%;margin: 0;padding: 0;}

设置地图

让我们创建一个使用本地GWC发布的地图。首先，我们初始化地图，并且设置地图在视窗的中心点和缩放级别，并添加底图图层：

var map = L.map('map', {
    minZoom: 3,
    maxZoom: 17,
    center: [32.550339, 106.114129],
    zoom: 4,
    attributionControl: false
});
var url = "http://a.tile.openstreetmap.org/{z}/{x}/{y}.png";
L.tileLayer(url).addTo(map);

现在你已经学会使用Leaflet创建一个地图应用。如果想要继续开发更多功能，请查看详细的API文档和示例。
